QT_BEGIN_NAMESPACE WebSocketTransport::WebSocketTransport ( QWebSocket *  socket)
explicit

QWebChannelAbstractSocket implementation that uses a QWebSocket internally.

The transport delegates all messages received over the QWebSocket over its textMessageReceived signal. Analogously, all calls to sendTextMessage will be send over the QWebSocket to the remote client.

Construct the transport object and wrap the given socket.

The socket is also set as the parent of the transport object.
